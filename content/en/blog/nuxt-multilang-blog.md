### Nuxt Multilanguage Blog

As you can see, I remade my blog, and added support for multi language, but in order to create it, I followed a few tutorials from the internet.

If you wish to make your blog as I did, I'll leave the links of the steps I followed to adding language support.

### Knowing the basics

If you wish to understand how the nuxt-content module works, I suggest you follow the official documentation from the Nuxt Docs.

[Creating A blog](https://nuxtjs.org/tutorials/creating-blog-with-nuxt-content/)

I recommend you follow the tutorial in practice, that way you understand how the content module works, and understand how the markdown metadata gets loaded.

The tutorial focuses on creating a basic blog, but not how to add multi language support, for that, you should follow the next blog.

[Dev.to - How to build a multi language blog with Nuxt](https://dev.to/astagi/how-to-build-a-jamstack-multi-language-blog-with-nuxt-js-3gah)

This post will show you how to add and work with il8n module to load and translate content.

With that post you can also access the example repository, it works well for basic tutorial, but for a better working example you can visit

[alvarosaburido-nuxt-il8n-blog](https://github.com/alvarosaburido/nuxt-i18n-blog)

This repo has better code, and much of my blog is based on the code presented in that repo.
