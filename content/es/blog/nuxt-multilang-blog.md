### Nuxt Blog Multilenguage

Como pueden ver, al rehacer mi blog agregue soporte multi lenguage, pero para poder crearlo, segui algunos tutoriales de internet.

Si deseas hacer un blog como yo lo hice, les dejo los links y los pasos que segui para agregar soporte de lenguajes.

### Conociendo lo basico

Si deseas entender como trabaja el modulo de nuxt-content, sugiero que sigan la documentacion oficial de Nuxt.

[Creating A blog](https://nuxtjs.org/tutorials/creating-blog-with-nuxt-content/)

Recomiendo que sigan el tutorial en practica, de esa forma pueden entender como trabaja el modulo de content, y entender como los metadatos de markdown se cargan.

El tutorial se enfoca en crear un blog basico, pero no para multi lenguaje, para eso ocupan leer el siguiente blog.

[Dev.to - How to build a multi language blog with Nuxt](https://dev.to/astagi/how-to-build-a-jamstack-multi-language-blog-with-nuxt-js-3gah)

Este post te ayudara a entender como trabaja el modulo il8n para traducir contenido.

Con este post tambien tienen acceso a un repositorio de ejemplo que funciona bien como tutorial basico, pero para entender mejor, ocupan visitar el siguiente repositorio.

[alvarosaburido-nuxt-il8n-blog](https://github.com/alvarosaburido/nuxt-i18n-blog)

Este repo tiene mejor codigo el cual utilize para mi blog.
